# 手牌スキャン＆修正ワークフロー

麻雀何切るアプリの問題データを、手牌画像から自動生成し、管理画面で修正するまでの手順をまとめたスキルです。

---

## 全体の流れ

```
画像フォルダ (samplequestions/)
       ↓ Claude API が画像を読み取り牌コードに変換
problems.json に自動書き込み
       ↓ 管理画面 (admin.html) で確認・修正
正しい problems.json の完成
```

---

## 事前準備（初回のみ）

### 1. Anthropic API キーの取得

1. `console.anthropic.com` にアクセスしてアカウント作成
2. 「Billing」でクレジットカードを登録・クレジット購入（最低 $5）
3. 「API Keys」→「Create Key」でキーを発行
4. 表示された `sk-ant-...` をコピー（画面を閉じると二度と見られないので注意）

### 2. 依存パッケージのインストール（初回のみ）

```powershell
cd "d:\ClaudeCode\麻雀何切るアプリ開発"
npm install
```

---

## 手順

### Step 1 — APIキーをセット

PowerShell を開き、以下を実行：

```powershell
$env:ANTHROPIC_API_KEY="sk-ant-ここにAPIキーを貼り付け"
```

> ⚠️ PowerShell を閉じると消えるので、毎回実行が必要です。

---

### Step 2 — 画像スキャン実行

Claude Code で以下のコマンドを呼び出す：

```
/scan-tiles
```

または引数でカテゴリーを絞ることもできます：

```
/scan-tiles 1_リーチ判断
```

**オプション：**

| コマンド | 動作 |
|---------|------|
| `/scan-tiles` | 未処理分のみスキャン（--resume） |
| `/scan-tiles 1_リーチ判断` | 指定カテゴリーのみスキャン |
| `/scan-tiles resume` | 前回の中断箇所から再開 |

スキャン中はプログレスバーで進行状況が表示されます：

```
[████████░░░░░░░░░░░░] 40% (171/428問)
カテゴリー: 9_くっつき形の選択1 (3/20問)
成功: 168問  スキップ: 3問  エラー: 0問
```

---

### Step 3 — 管理画面で修正

開発サーバーを起動：

```powershell
npm run dev
```

ブラウザで管理画面を開く：

```
http://localhost:5173/admin.html
```

#### 管理画面の使い方

1. **左サイドバー**でカテゴリーを選択
2. **問題番号**をクリック
3. 右側に問題画像と現在の手牌が表示される

**手牌の修正：**

| 操作 | 内容 |
|------|------|
| 手牌の牌をクリック | その牌を削除 |
| 牌パレットの牌をクリック | 手牌に追加 |

**正解牌の設定：**
- 「正解牌」エリアの牌をクリック → 緑でハイライトされる

**ドラの設定：**
- 「変更する」ボタン → 全牌パレットから選択
- 「なし」ボタン → ドラなしに設定

4. **「保存」ボタン**を押すと `problems.json` に即反映

---

### Step 4 — 画像ファイルのコピー（カテゴリー追加時）

新しいカテゴリーの問題を管理画面で表示するには、画像を `public/` にコピーする必要があります。

Claude Code に以下のように依頼：

```
〇〇カテゴリーの画像を public にコピーしてください
```

---

## エラーが出た場合

スキャン中にエラーになった問題は `scripts/progress.json` の `errors` フィールドに記録されます。

```json
{
  "errors": [
    { "key": "4_2メンツ形の選択1/5.jpg", "error": "JSONが見つかりません" }
  ]
}
```

エラーになった問題だけ再スキャンするには：

```
/scan-tiles 4_2メンツ形の選択1
```

## ファイル構成

```
.claude/commands/scan-tiles.md   ← このスキルの定義
scripts/
  generate-problems.mjs          ← スキャンスクリプト本体
  progress.json                  ← 処理済み記録（自動生成）
src/
  data/problems.json             ← 問題データ（スキャン結果＋修正済み）
  admin/                         ← 管理画面コンポーネント
samplequestions/                 ← 元の手牌画像（カテゴリー別）
public/samplequestions/          ← Webアプリが参照する画像置き場
```
